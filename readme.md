Для запуска Katalog необходимо:

- клонировать репозиторий;

- установить зависимости из package.json;

- запустить сервер с помощью Node.js (для реализации $routeProvide Angular.js);

- для запуска сервера необходимо использовать файл app.js;

Web-приложение будет доступно по адресу http:// localhost:3000
